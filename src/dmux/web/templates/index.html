Setup details

Plugins are listed in a generated file and cloned under ~/.tmux/plugins/. Bootstrap appends a source-file hook to ~/.tmux.conf (see exact path below). To use a different file (for example ~/.config/tmux/tmux.conf), set DMUX_TMUX_CONF before starting dmux ui.

Freed-Wu/tmux-status-bar notes

dmux writes set -g status-left / status-right lines before run tpm so TPM’s status-bar.tmux sees them. The plugin requires tmux-powerline-compiler on PATH — without it the plugin silently overwrites both options with an empty string on every source-file (blank status bar). Build it with brew install xmake flex bison && cd ~/.tmux/plugins/tmux-status-bar && xmake (or cpan tmux-status-bar for the perl variant) — see install.

tmux-status-bar wizard

Configure JIT status strings for Freed-Wu/tmux-status-bar (#{status-left:…}, #{status-right:…}, #{window-status-current-format-…:…}). Add the plugin to your list first. Text fields may use tmux formats such as #{session_name}, #{window_name}, or other plugins’ #{} helpers.

Why you don’t see a “status bar” here: the dmux browser UI never draws tmux’s status line — only sessions/panes. The real bar appears in a terminal after tmux attach (or new tmux). If it’s missing there too, check tmux show -g status (must be on), reload config, and ensure Freed-Wu/tmux-status-bar is built/loaded so #{status-left:…} is expanded instead of shown raw or empty. If you see a long number and ;white,colour04,…, tmux is expanding a raw % in status-left/status-right as strftime (e.g. %s → Unix time). Remove those % sequences or use only #{…} / #(…); then Apply config for Freed-Wu and Source in tmux.

          Pane appearance
        

        

          Text / background are applied in tmux with
          set-option -p -t <pane> window-style and
          window-active-style (same as running tmux from your shell — use
          dmux ui -S if you use a non-default socket). Use tmux colour syntax:
          #rrggbb, colour0–colour255, or names like
          red — not rgb(r,g,b) (commas break tmux styles).
          Font applies only to the dmux web tile; terminals do not support per-pane font via tmux.
