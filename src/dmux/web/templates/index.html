Rename session

Same as dmux rename <old> <new> — tmux session name must stay unique.

Pane appearance

Text / background are applied in tmux with set-option -p -t <pane> window-style and window-active-style (same as running tmux from your shell — use dmux ui -S if you use a non-default socket). Use tmux colour syntax: #rrggbb, colour0–colour255, or names like red — not rgb(r,g,b) (commas break tmux styles). Font applies only to the dmux web tile; terminals do not support per-pane font via tmux.

New session

Starts a detached tmux session you can attach to from a terminal.
